# 🚗 Real-Time Parking Occupancy Detection Using OpenCV

## 📌 Project Description
This project presents a **real-time parking occupancy detection system** developed using Python and OpenCV. The system is designed to efficiently monitor parking spaces and provide live updates on available and occupied slots. It addresses common urban challenges such as traffic congestion and difficulty in finding parking.

The system uses computer vision techniques to analyze video feeds from cameras and detect vehicles dynamically, helping users and parking administrators make informed decisions.

## 🎯 Objectives

- Reduce traffic congestion caused by parking search  
- Provide accurate and dynamic parking availability  
- Build a scalable and efficient smart parking solution

## 🧠 System Working
The system works in three main stages:

1. **Vehicle Detection**  
   Detects vehicles using computer vision algorithms  

2. **Classification**  
   Identifies whether the vehicle is parked or moving  

3. **Tracking**  
   Tracks vehicle movement and updates parking status in real-time
